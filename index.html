How It Works

1

Install & Run

Download and install TranslateSutra. A floating button appears on your desktop.

2

Select or Copy Text

Select text in any application or copy it to your clipboard.

3

Press Global Hotkey

Press Ctrl+Shift+T (or Cmd+Shift+T on Mac) from anywhere.

4

Get Instant Translation

A beautiful popup shows the translation. Copy, pin, or close as needed.
