## Step 1: Upload Files to GitHub

1. **Copy the `root` folder contents** to your GitHub repository root
2. **Push to GitHub** (main branch)

## Step 2: Enable GitHub Pages

1. Go to: `https://github.com/hieabhi/translatesutra/settings/pages`
2. **Source**: "Deploy from a branch"
3. **Branch**: `main` 
4. **Folder**: `/ (root)`
5. Click **Save**

## Step 3: Create Release

1. Go to: `https://github.com/hieabhi/translatesutra/releases`
2. Click **"Create a new release"**
3. **Tag**: `v1.0`
4. **Title**: `TranslateSutra v1.0 - Floating Desktop Translator`
5. **Upload**: `TranslateSutra-v1.0-Portable.zip`
6. Click **"Publish release"**

## Step 4: Your Live Website

After setup, your website will be live at:
**https://hieabhi.github.io/translatesutra**
